# 119 · manual-form-fields — 무ISBN 폼 필드 구성 점검 (사용자 아침 제보 4·5번)

배경: ISBN 경로 대비 페이지·복본수 부재 + 서지 관례상 부제 위치 + 분류 코드 예시 부재.

조사 결론(선반영):
- **페이지수 — 추가하지 않음**: 03_TITLES에 page_count 컬럼이 없고(스키마 동결), 페이지는
  21_BOOK_CACHE(isbn13 키)에만 산다 — ISBN 없는 서지는 저장처 자체가 없다. 조용히 버려질
  필드를 만드는 건 금물. (향후 TITLES 스키마 확장은 별도 결정 사항.)
- **복본수 — 폼에 추가하지 않음**: registerTitle_은 1권 생성이 계약이고, 추가 복본은 완료
  카드의 「복본 추가 발급」(부분 실패 UX 포함)이 이미 정답 경로 — 같은 일 두 입구는 혼란.

할 일(잔여): 부제를 서명 바로 아래로(서명–부제 짝), 분류 코드 placeholder(예: 800, 813),
발행년 inputmode는 이미 numeric(확인됨). 완료 카드 복본 발급 안내는 기존 그대로.

완료 조건: 재캡처, 전 게이트, e2e.
