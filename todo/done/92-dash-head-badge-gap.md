# 92 · dash-head-badge-gap — 대시보드 패널 제목·배지 밀착 (시각 감사 1R)

배경(증빙 /tmp/vis/d1-dashboard.png): 기저층 패널 h2가 `{제목}{샘플배지}` 인라인 연결이라
「예약 도착샘플 데이터」처럼 제목과 배지가 붙어 보인다(최근 처리도 동일).

할 일: `.dash-panel h2`를 flex(gap 8, wrap)로 — 배지 유무와 무관하게 제목 줄 정렬 유지.
다른 배지 사용처(설정 헤드 등)는 이미 gap 컨테이너라 건드리지 않는다.

완료 조건: 캡처 1장, 전 게이트.
