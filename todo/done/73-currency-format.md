# 73 · currency-format — 금액 표기 통일 (Intl KRW)

배경: 연체료·변상금이 화면마다 날수/원 표기가 제각각일 수 있다 — 리포트·미변상·정책·학생방.

할 일: formatKRW 유틸(Intl.NumberFormat ko-KR) 신설, 전 사용처 치환(en 로케일은 KRW 표기 유지).
0원·미정 표기 규칙 통일.

완료 조건: 사용처 목록 기록, 전 게이트.

## 이행 노트
- 실태: 동일 구현의 formatCurrency가 뷰 4곳에 복제(중복 금지 위반 상태) — 표기는 이미 KRW
  통일이었으므로 이 항목의 실작업은 "단일 원천화": i18n/format.ts formatKRW 신설, 4곳 치환.
- 설정의 정책 표(13_POLICIES 미러)는 원값 유지 — 시트 대조 용도라 가공하지 않는다(기록).
- 학생 표면·기타 화면에 비포맷 금액 노출 없음(전수 grep).
