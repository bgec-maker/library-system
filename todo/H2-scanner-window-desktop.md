# H2 · 데스크톱 스캐너 창 (ADR-026 — ADR-018 일부 개정)
## 결정 (완료 시 docs/DECISIONS.md에 아래 ADR-026 그대로 추가)
> ## ADR-026 · 데스크톱 스캐너 = 창 (2026-07-16)
> **결정** — "스캐너는 창이 아니다"(ADR-018 부수결정)를 개정: 온디맨드(ADR-020) 세계에선 **창 = 카메라 세션**이 더 단순하다. 창 열기=카메라 시작, 닫기=종료, 연속 모드 핀 중엔 최소화해도 유지. 도크 위젯은 상태점+열기 버튼으로 축소.
> **왜** — 구 결정은 상시 구동 전제("창을 닫아도 카메라는 산다")였다. 온디맨드에선 그 분리가 오히려 상태를 숨긴다. 사용자 요구: 프리뷰를 제대로 된 창으로.
## 구현
- shells/desktop/ScannerWindow (뷰 아님 — 카메라는 셸 관심사, views/ 린트 경계 밖) · single
- 내용: 프리뷰 + 조준 프레임(H1과 동일 규격 공유 컴포넌트) + "→ 수신 창: {제목}" + 연속 핀 + 유휴 카운트다운
- 열기: 도크 위젯 클릭 · 단축키 S · 스캔 필요 뷰의 시작 버튼 → 모두 이 창 오픈
- 닫기=카메라 off (연속 핀 시 확인 1회) · 최소화=유지(러시아워)
